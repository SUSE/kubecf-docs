A Kubernetes Native Distribution of Cloud Foundry
KubeCF is a distribution of Cloud Foundry Application Runtime (CFAR) for Kubernetes. It gives developers the productivity they love from Cloud Foundry and allows platform operators to manage the infrastructure abstraction with Kubernetes tools and APIs.
